# 🍽️ FoodShare AI – Smart Food Waste Sharing Platform

FoodShare AI is a full-stack web application that reduces food waste by connecting food donors, NGOs, and volunteers through AI-powered matching and real-time tracking.

## 🚀 Features

- 🍱 Food Donation Management
- 🤖 AI-Based Food Matching
- 🏢 Restaurant & NGO Dashboards
- 🚚 Volunteer Pickup Tracking
- 📍 Real-Time Location Tracking
- 📊 Analytics Dashboard
- 🔔 Notifications
- 🌙 Dark & Light Mode
- 📱 Responsive Design

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- TypeScript
- Framer Motion
- Chart.js

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- REST API

### Database
- MySQL

## 📂 Project Structure

```
foodshare-ai/
├── frontend/
├── backend/
└── database/
```

## ⚙️ Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
mvn spring-boot:run
```

## 🎯 Objective

To reduce food waste by enabling smart food donation and efficient distribution using AI.
